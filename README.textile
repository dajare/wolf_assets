h3. What?

"Mephisto":http://www.mephistoblog.com/ style asset management for "Frog CMS":http://www.madebyfrog.com/. You can drag thumbnail to body text to insert full URL to image.

!http://www.appelsiini.net/assets/2008/12/11/assets_page.png!

Includes simple interface for uploading images.

!http://www.appelsiini.net/assets/2008/12/11/assets_admin.png!

*Code is pretty much beta quality. Wrote in one afternoon because I needed it. Configuration options will follow later.*

h3. Install

Make assets folder into your document root. 

<pre>
cd /path/to/document/root
mkdir assets
chmod a+w assets
</pre>

Copy plugin files to _frog/plugins/assets/_ folder.

<pre>
cd frog/plugins/
git clone git://github.com/tuupola/frog_assets.git assets
</pre>

You also need Peter Gassners excellent "Image Resize":http://github.com/naehrstoff/image_resize/ plugin.

<pre>
cd frog/plugins/
git clone git://github.com/naehrstoff/image_resize.git image_resize
</pre>

Go to admin and enable Assets and Image Resize plugins. Last thing is to add jQuery to backend template. Currently you need to do this by hand.

<pre>
cd frog/app/backend/layouts/
mate backend.php
</pre>

After

<pre>
<script type="text/javascript" charset="utf-8" src="javascripts/control.textarea.js"></script>
</pre>

add following lines:

<pre>
<script type="text/javascript" src="http://ajax.googleapis.com/ajax/libs/jquery/1.2.6/jquery.min.js"></script>
<script>
  jQuery.noConflict();
</script>
</pre>

Now you are set to go.
