h3. What?

"Mephisto":http://www.mephistoblog.com/ style asset management for "Frog CMS":http://www.madebyfrog.com/. You can drag thumbnail to body text to insert full URL to image.

!http://www.appelsiini.net/assets/2008/12/22/assets_page.png!

Includes simple interface for uploading images.

!http://www.appelsiini.net/assets/2009/2/5/asset_uploaded.png!

h3. Install

Make assets folder into your document root. 

<pre>
cd /path/to/document/root
mkdir assets
chmod a+w assets
</pre>

Copy plugin files to _frog/plugins/assets/_ folder.

<pre>
cd frog/plugins/
git clone git://github.com/tuupola/frog_assets.git assets
</pre>

You also need Peter Gassners excellent "Image Resize":http://github.com/naehrstoff/image_resize/ plugin and my "jQuery":http://github.com/tuupola/frog_jquery and "jQuery UI":http://github.com/tuupola/frog_jquery_ui plugins.

<pre>
cd frog/plugins/
git clone git://github.com/naehrstoff/image_resize.git image_resize
</pre>

<pre>
cd frog/plugins/
git clone git://github.com/tuupola/frog_jquery.git jquery
</pre>    

<pre>
cd frog/plugins/
git clone git://github.com/tuupola/frog_jquery_ui.git jquery_ui
</pre>    


Go to admin and enable Assets and Image Resize, jQuery and jQuery UI plugins. Now you are set to go.
