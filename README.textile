h3. What?

"Mephisto":http://www.mephistoblog.com/ style asset management for "Frog CMS":http://www.madebyfrog.com/. You can drag thumbnail to body text to insert full URL to image.

!http://www.appelsiini.net/assets/2008/12/11/assets_page.png!

Includes simple interface for uploading images.

!http://www.appelsiini.net/assets/2008/12/11/assets_admin.png!

*Code is pretty much beta quality. Written in one afternoon because I needed it. Configuration options will follow later.*

h3. Install

Make assets folder into your document root. 

<pre>
cd /path/to/document/root
mkdir assets
chmod a+w assets
</pre>

Copy plugin files to _frog/plugins/assets/_ folder.

<pre>
cd frog/plugins/
git clone git://github.com/tuupola/frog_assets.git assets
</pre>

You also need Peter Gassners excellent "Image Resize":http://github.com/naehrstoff/image_resize/ plugin and my "jQuery":http://github.com/tuupola/frog_jquery plugin.

<pre>
cd frog/plugins/
git clone git://github.com/naehrstoff/image_resize.git image_resize
</pre>

<pre>
cd frog/plugins/
git clone git://github.com/tuupola/frog_jquery.git jquery
</pre>    

Go to admin and enable Assets and Image Resize and jQuery plugins. Now you are set to go.
